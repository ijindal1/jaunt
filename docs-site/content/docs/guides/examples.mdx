If you want the hackathon story, start with `jaunt-examples/`.

These demos are designed for the “wow gap”: the spec is a small contract, but the generated output is lots of boring glue, edge cases, and tests.

Important: running these will call the OpenAI API and spend tokens (set `OPENAI_API_KEY`).

## Hackathon-Ready Demos (`jaunt-examples/`)

### JWT Auth (The “Wow Gap” Demo)

This demo is intentionally tiny on spec and big on real correctness work: base64url encoding, HMAC signing, strict parsing, and expiry checks.

`jaunt-examples/jwt_auth/src/jwt_demo/specs.py` (excerpt):

```python
from datetime import timedelta

import jaunt
from pydantic import BaseModel


class Claims(BaseModel):
    """Decoded token payload."""

    sub: str  # subject (user id)
    iat: float  # issued-at (unix timestamp)
    exp: float  # expiry (unix timestamp)


@jaunt.magic()
def create_token(
    user_id: str,
    secret: str,
    *,
    ttl: timedelta = timedelta(hours=1),
) -> str:
    """
    Create an HS256-signed JWT.

    Structure: base64url(header) . base64url(payload) . base64url(signature)

    - Use HMAC-SHA256 with `secret` as the key.
    - base64url encoding must omit padding ("=" characters).
    """
    raise RuntimeError("spec stub (generated at build time)")
```

Run it from the repo root:

```bash
# 1) Generate implementation modules for @jaunt.magic specs.
uv run jaunt build --root jaunt-examples/jwt_auth

# 2) Generate pytest tests for @jaunt.test specs and run them.
PYTHONPATH=jaunt-examples/jwt_auth/src uv run jaunt test --root jaunt-examples/jwt_auth
```

Skills auto-generation is part of the story here: the spec imports `pydantic`, so `jaunt build` writes a generated skill at:

`jaunt-examples/jwt_auth/.agents/skills/pydantic/SKILL.md`

`jaunt-examples/jwt_auth/src/jwt_demo/__generated__/specs.py` (excerpt):

```python
def create_token(
    user_id: str,
    secret: str,
    *,
    ttl: timedelta = timedelta(hours=1),
) -> str:
    if not isinstance(user_id, str) or user_id == "":
        raise ValueError("user_id must be non-empty")
    _ = _validate_ttl(ttl)

    iat = _now_iat()
    exp = iat + int(ttl.total_seconds())

    header = {"alg": "HS256", "typ": "JWT"}
    payload = {"sub": user_id, "iat": iat, "exp": exp}

    header_b = json.dumps(header, separators=(",", ":"), sort_keys=True).encode("utf-8")
    payload_b = json.dumps(payload, separators=(",", ":"), sort_keys=True).encode("utf-8")

    header_b64 = _b64url_encode(header_b)
    payload_b64 = _b64url_encode(payload_b)
    signing_input = f"{header_b64}.{payload_b64}".encode("ascii")

    sig = hmac.new(secret.encode("utf-8"), signing_input, hashlib.sha256).digest()
    sig_b64 = _b64url_encode(sig)

    return f"{header_b64}.{payload_b64}.{sig_b64}"
```

Pytest generated by `jaunt test`:

`jaunt-examples/jwt_auth/tests/__generated__/specs.py` (excerpt):

```python
def _claims_get(claims, key: str):
    if hasattr(claims, key):
        return getattr(claims, key)
    if isinstance(claims, dict) and key in claims:
        return claims[key]
    raise AssertionError(f"Claims missing {key!r}: {claims!r}")
```

`jaunt-examples/jwt_auth/tests/__generated__/specs.py` (excerpt):

```python
def test_roundtrip_create_and_verify() -> None:
    api = _load_api()
    token = api.create_token("user-42", "s3cret")
    claims = api.verify_token(token, "s3cret")
    assert _claims_get(claims, "sub") == "user-42"
    assert _claims_get(claims, "exp") > _claims_get(claims, "iat")
```

## Older Runnable Examples (`examples/`)

The `examples/` folder contains older small Jaunt projects you can run end-to-end.

From the repo root:

```bash
uv run python examples/run_example.py slugify test
uv run python examples/run_example.py lru test
uv run python examples/run_example.py dice test
```

On-the-fly demo (scaffolds a temp project, runs `jaunt build`, optionally `jaunt test`):

```bash
uv run python examples/demo_on_the_fly.py --test --keep
```
