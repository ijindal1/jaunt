The `examples/` folder contains small Jaunt projects you can run end-to-end. Each one has:

- `src/<pkg>/specs.py`: `@jaunt.magic` spec stubs (implementation intent)
- `tests/specs.py`: `@jaunt.test` specs (test intent)
- generated outputs under `src/<pkg>/__generated__/...` and `tests/__generated__/...`

Important: running these will call the OpenAI API and spend tokens.

## Run An Example

From the repo root:

```bash
.venv/bin/python examples/run_example.py slugify test
.venv/bin/python examples/run_example.py lru test
.venv/bin/python examples/run_example.py dice test
```

On-the-fly demo (scaffolds a temp project, runs `jaunt build`, optionally `jaunt test`):

```bash
.venv/bin/python examples/demo_on_the_fly.py --test --keep
```

## Slugify (01_slugify)

`examples/01_slugify/src/slugify_demo/specs.py` (excerpt):

```python
import jaunt


@jaunt.magic()
def slugify(title: str) -> str:
    """
    Convert a human title into a URL-safe slug.

    Rules:
    - Trim surrounding whitespace.
    - Lowercase.
    - Replace any run of non-alphanumeric characters with a single "-".
    - Strip leading/trailing "-".
    - Must return a non-empty string.
    """
    raise RuntimeError("spec stub (generated at build time)")
```

`examples/01_slugify/tests/specs.py` (excerpt):

```python
import jaunt


@jaunt.test()
def test_slugify_basic_cases() -> None:
    from slugify_demo import slugify

    assert slugify("  Hello, World!  ") == "hello-world"
```

```python
import jaunt


@jaunt.magic()
class LRUCache:
    """A small least-recently-used (LRU) cache keyed by strings."""

    def __init__(self, capacity: int) -> None:
        """capacity must be >= 1 (otherwise raise ValueError)."""
        raise RuntimeError("spec stub (generated at build time)")
```

`examples/02_lru_cache/tests/specs.py` (excerpt):

```python
import jaunt


@jaunt.test()
def test_invalid_capacity_raises() -> None:

```

## Dice Roller (03_dice_roller)

This example uses an injected RNG for determinism.

```python
import random
import jaunt


@jaunt.magic()
def roll(expr: str, *, rng: random.Random) -> int:
    """Roll a dice expression using a provided RNG and return the total."""
    raise RuntimeError("spec stub (generated at build time)")
```

### Generated Output (Excerpt)

After you run `examples/run_example.py dice build`, Jaunt produces a real implementation module:

```python
# This file was generated by jaunt. DO NOT EDIT.
# jaunt:kind=build
# jaunt:source_module=dice_demo.specs

import re

_DICE_RE = re.compile(r"^\\s*(?:(?P<count>\\d+)?d(?P<sides>\\d+))(?:(?P<sign>[+-])(?P<bonus>\\d+))?\\s*$")

def roll(expr: str, *, rng):
    count, sides, bonus = parse_dice(expr)
    total = 0
    for _ in range(count):
        total += rng.randint(1, sides)
    return total + bonus
```

And it writes executable pytest tests derived from your `@jaunt.test` intent:

```python
# This file was generated by jaunt. DO NOT EDIT.
# jaunt:kind=test
# jaunt:source_module=tests.specs

def test_roll_is_deterministic_with_seeded_rng() -> None:
    rng = random.Random(0)
    assert roll("2d6+3", rng=rng) == 11
```

## Output Locations

Generated outputs are written inside each example project:

- `src/<pkg>/__generated__/...` (implementations)
- `tests/__generated__/...` (pytest tests)

Review the generated code before relying on it in real projects.

## Pydantic Validation (04_pydantic_validation)

This example showcases **auto-generated PyPI skills** during `jaunt build`.

It imports `pydantic` in the spec module, so Jaunt will (best-effort) generate a skill at:

`examples/04_pydantic_validation/.agents/skills/pydantic/SKILL.md`

`examples/04_pydantic_validation/src/pydantic_demo/specs.py` (excerpt):

```python
import jaunt
import pydantic


@jaunt.magic()
class User(pydantic.BaseModel):
    model_config = pydantic.ConfigDict(extra="forbid")
```

Run it from the repo root:

```bash
.venv/bin/python examples/run_example.py pydantic test
```
