This repo’s hackathon demos live in `jaunt-examples/`: small consumer projects that run Jaunt as an external tool (like a real repo would).

## The “Wow Gap”

The spec is short and readable, but the generated output is where the boring correctness work shows up: strict parsing, edge cases, glue code, and runnable tests.

The headline demo is JWT auth:

- Spec: `jaunt-examples/jwt_auth/src/jwt_demo/specs.py`
- Build output: `jaunt-examples/jwt_auth/src/jwt_demo/__generated__/...`
- Test output: `jaunt-examples/jwt_auth/tests/__generated__/...`

## Run It

From the repo root (calls OpenAI, spends tokens):

```bash
uv run jaunt build --root jaunt-examples/jwt_auth
PYTHONPATH=jaunt-examples/jwt_auth/src uv run jaunt test --root jaunt-examples/jwt_auth
```
