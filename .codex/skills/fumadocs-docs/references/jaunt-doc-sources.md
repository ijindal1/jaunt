# Jaunt Doc Sources (Repo Index)

Use this as a map of existing documentation sources in this repository when building a Fumadocs site.

## Primary docs

- `README.md`: High-level overview.
- `DOCS.md`: Full repo tour, quickstart, configuration (`jaunt.toml`), specs (`@jaunt.magic`), test specs (`@jaunt.test`), CLI usage.

## Architecture / design notes (link instead of duplicating)

- `TASK-*.md`: Focused design and implementation notes.
- `jaunt_tdd_plan.md`: Larger implementation plan and behavior notes.
- `examples/`: Runnable example projects demonstrating Jaunt features.

## Code references

- `src/jaunt/`: Implementation (use it to confirm real API/CLI names).
- `tests/`: Behavioral truth source; prefer doc statements that match tests.
